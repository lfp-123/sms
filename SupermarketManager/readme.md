# SSM+JSP+Shiro+Mysql实现超市管理系统

## 功能需求

1. 人力资源管理
	- 提供企业员工的信息管理
	- 员工部门的信息管理
	- 企业员工考勤的信息管理
	- 用户权限的管理
2. 公司财务部
	- 员工工资信息管理
	- 商品销售业绩管理
	- 商品采购费用信息管理
	- 用户密码修改
3. 商品采购部
	- 商品类型信息管理
	- 商品信息管理
	- 供应商厂家信息管理
	- 采购联系人信息管理
	- 采购信息管理
4. 商品销售部
	- 商品所有信息显示
	- 商品销售管理
5. 用户权限修改
	- 用户密码修改
	- 用户权限修改
6. 个人信息管理
	- 个人详细信息查询
	- 个人工资信息查询
	- 个人考勤信息查询
	- 个人密码修改
